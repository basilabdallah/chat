## 🚀 Quick Start

### 3 Commands to Get Running

```bash
# 1. Install
npm install

# 2. Configure (add your Supabase credentials)
cp .env.local.example .env.local

# 3. Run
npm run dev
```

**Don't forget**: Set up Supabase database first!
